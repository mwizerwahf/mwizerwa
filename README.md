# MwizerwaSys Solutions - Professional Technology Services

![MwizerwaSys Solutions Logo](assets/icons/logo.svg)

## Overview

MwizerwaSys Solutions is a professional technology services company specializing in comprehensive tech solutions for both residential and business clients. Our services include CCTV installation, network setup, device repair, and website development.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Service Showcase**: Highlighting key services with detailed descriptions
- **Portfolio Section**: Display of recent projects and work samples
- **Testimonials**: Client feedback and ratings
- **Contact Form**: Easy-to-use contact form for service inquiries
- **Interactive Elements**: Animated call-to-action buttons and hover effects

## Services

1. **CCTV Installation**
   - Professional camera placement and configuration
   - Remote monitoring setup
   - 24/7 security solutions

2. **Network Setup & Security**
   - Wired and wireless network installation
   - Network security solutions
   - Business network infrastructure

3. **Device Repair**
   - Laptop and computer repair
   - Smartphone and tablet repair
   - Hardware and software troubleshooting

4. **Website Development**
   - Responsive website design
   - E-commerce solutions
   - Web application development

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **CSS Framework**: Custom CSS with Flexbox and Grid
- **Icons**: Custom SVG and Font Awesome
- **Animations**: CSS Animations and Transitions
- **Form Handling**: JavaScript Form Validation

## Getting Started

1. **Clone the repository**
   ```bash
   git clone [repository-url]
   ```

2. **Open in your preferred code editor**

3. **Launch the website**
   - Open `index.html` in your web browser
   - Or use a local server (e.g., Live Server in VS Code)
